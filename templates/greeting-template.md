## 技术栈描述规则

| 技能 | 描述 |
|------|------|
| React/Vue | 熟练掌握 |
| TypeScript | 熟练掌握 |
| ThreeJs/CesiumJs/Echarts | 熟练掌握 |
| Ant Design/ElementUI | 熟练掌握 |
| 微信小程序/支付宝小程序 | 有完整开发经验 |
| Node.js/Python | 熟悉 |

- ❌ 专家

## 示例

### 示例 1：React 岗位

```
您好，我是小🦞虾，黎哥的 AI 助手。

我仔细分析了这个金融 React 前端开发岗位（18-22K）的要求，并结合黎哥的经历进行了匹配：

【匹配点】
1. 经验匹配：11 年工作经验，6 年软件开发经验（符合 5-10 年要求）
2. 技术栈匹配：
   - 熟练掌握 React 框架，有实际项目经验
   - 熟练掌握 Ant Design 等 UI 库
   - 熟悉 Webpack 等前端构建工具
3. 架构能力：
   - 有前端架构设计与选型经验
   - 有组件化、模块化开发完整经验
4. 团队协作：
   - 现任前端开发组长，与 UI、后端团队高效协同

【核心优势】
- React 专家：熟练掌握 React 及 Ant Design 技术栈
- 架构经验：有前端架构设计、组件化开发经验
- 团队管理：现任前端开发组长，有良好的团队合作精神
- 学习能力：能快速定位排查项目紧急 BUG

作为黎哥的 AI 助手，我觉得这个岗位与黎哥的经历非常匹配，尤其是 React、Ant Design 和架构经验方面，希望能有机会进一步沟通，谢谢！
```
